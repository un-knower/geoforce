超图云API

地图API是支持各种客户端，包括Javascript、富客户端、三维插件胖客户端等，它能够帮助您在网站中构建功能丰富、交互性强的地图应用程序。地图API不仅包含构建地图的基本接口，还提供了诸如本地搜索、路线规划等数据服务，可根据自己的需要进行选择。

功能简介

基本地图操作：拖动、平移、缩放等；

地图控件操作：比例尺、导航、缩放等控件的显示/隐藏、对齐；

地图绘制操作：画圆、画矩形、画线、画多边形等；

地图标注操作：图片、文本等标记的显示/隐藏、拖拽、动画等；

矢量编辑操作：支持线、面节点编辑和拖曳；

本地搜索：提供指定地区的位置检索服务，比如在北京搜索“酒店”；

驾车导航：提供驾车出行方案的检索服务；

key的获取与使用

使用地图 API 可在您自己的网页上内嵌超图在线地图。一个地图 API Key 只对您网络服务器上的一个“目录”有效。

要获得 API，您必须有超图地理信息云门户的帐户。向超图地理信息云门户提供您的帐户信息、服务网址并同意使用条款后，我们会向您发送一个由超图地理信息云门户系统自动分配的 Key ，该 Key 由字母数字组成，只与您的超图地理信息云门户帐户和服务网址相关。如地图 API 文档所述，您的服务必须使用含此 Key 的 HTML ”script”标记导入 超图地理信息云门户地图 API，否则超图地理信息云门户会因无效 Key 或无效网址而拦截请求。

地图服务是指提供地图数据的接口，如 Poi检索、路线计算服务、公交线路详情、公交站点详情等。

  • POI检索服务 POISearch
  • 路线分析 TransportationAnalyst

POI检索服务 SuperMap.OSP.Service.POISearch

提供指定地区的位置检索服务。你可以通过 SuperMap.OSP.Service.POISearch 对象检索POI数据。此对象与SuperMapCloud地图API Search Service 进行通信，该服务将接收检索请求并传回计算结果。

通过指定要检索的关键字文本字符串("如酒店")，要检索的地理区域范围（当前地图显示范围）来查找POI数据。检索服务返回的是多个 SuperMap.OSP.UI.POI 对象，并且能够分页获取。

检索请求 SearchRequest

SuperMapCloud地图 API 需要调用外部服务器，因此，对检索服务的访问是异步的。为此，您需要传递一个回调方法，以在请求完成后执行。此回调方法将会对结果进行处理。请注意，检索服务以 SearchResult 对象的形式传回Poi 数据,多个poi对象被包含在SearchResult对象的 records[] 属性中。

要检索Poi数据。需要创建一个类型为 SuperMap.OSP.Service.POISearch 的对象，并调用 POISearch.getPOIsByGeometry() 向检索服务发送一条请求，方法是为其传递三个参数分别是: SuperMap.OSP.Service.GetPOIsByGeometryParam 参数对象，回调方法，错误处理函数。

参数说明：

  1. scales：获取或设置地图比例尺数组。该属性与分辨率数组 resolutions 属性只需设置其一即可。
  2. url：图层所在的服务地址。
  3. mapName：获取或设置请求服务的地图名称。必设属性。
  4. mapServiceAddress：获取或设置地图 GIS 服务地址。例如 192.168.11.11，注意：无须在 IP 值前加 “HTTP：//”。必设属性。
  5. mapServicePoint：获取或设置地图服务的端口号。必设属性。

常见问题

什么是地图API

    SuperMapCloud 地图API是由超图提供的地理位置服务接口，应用程序接口由JavaScript语言编写的，它能够帮助您在网站中方便的搭建基于地图应用程序。 SuperMapCloud 地图开放API提供了：构建地图的基本接口、本地搜索接口等，您可以根据自己的需要进行选择。

地图API提供了哪些功能

    SuperMapCloud 地图提供了基本的地图构建；地点搜索；地址解析；反地址解析；公交、驾车路线搜索；绘制点线面等丰富的地图服务，具体的内容您可以参考 示例 或 详细阅读 参考手册 进行进一步的了解。

坐标体系是否遵循国家对地理信息保密要求

    SuperMapCloud 地图API对外接口的坐标系，都是经过国家测绘局加密处理，符合国家测绘局对地理信息保密要求。
