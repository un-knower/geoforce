丰富的地理数据内容

  • 基础地图：基础电子地图、九级导航路网、千万级POI点。
  • 专题地图：自然地理、社会经济等专业权威的专题地图。
  • 其他地图：遥感、建筑物多边形、三维等数据。

专业的地理信息服务

  • 地址匹配：输入地址，自动地址匹配，在地图上标明位置及相关空间信息。
  • POI搜索：POI定位、搜索，周边POI的查询支持用户自定义标注POI点。
  • 距离测算：规划起始点的最短路径，自动测算距离长度。

便捷的产品服务模式

  • 在线调用：访问超图云，即可在线体验，同时可在线调用API进行二次开发。
  • 离线部署：对有保密需求、不能联网或有发布地图服务需求的用户可离线部署。

SuperMap iData优势

  • 完美结合：无需繁琐的处理工序，与SuperMap平台软件完美结合。
  • 拿来即用：地图加工处理更新一条龙，用户直接获取专业化的地图数据成果。
  • 轻松发布：离线数据部署只需四步即可完成，无需解压，一键发布地图服务。
