地图慧在线地图服务http://t0.supermapcloud.com/FileService/image已经升级为 http://t2.supermapcloud.com/FileService/image，该地图服务的缓存从v3.0升级至v5.0。

http://t0.supermapcloud.com/FileService/image服务将于2016年9月12日停止，请尽快升级。

iClient 6R for JavaScript地图升级指南

iClient 7C or JavaScript、iClient 8C for JavaScript版本默认使用v5.0地图缓存服务，客户端无需升级。较早版本iClient 6R for JavaScript版本默认使用v3.0地图缓存服务，客户端需要相应升级。升级步骤：

1 在页面引用iClient for JavaScript的js之后，再引入dituhui.cloudlayer.js，

<script type="text/javascript" src="libs/SuperMap.Include.js"></script>

<script href="text/javascript" src="../dituhui.cloudlayer/dituhui.cloudlayer.js"></script>

2 地图初始化添加底图

var layer = new SuperMap.Layer.CloudLayer();

map.addLayer(layer);
